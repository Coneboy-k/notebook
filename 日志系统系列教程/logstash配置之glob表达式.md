# 概述
`glob`表达式是正则表达式的简化版,用于匹配文件路径.
# 模式
* `*`

    匹配任意的文件,比如"*.conf,*auto*"

* `**`

    递归匹配,遍历子文件夹

* `?`

    匹配任何一个字符

* `[set]`

    匹配存在在这个集合中的字符,[^set]匹配不存在

* `{p,q}`

    匹配任何`p`,`q`,类似`a|b`

* `\`

    转义下一个字符

# 例子
```
"/path/to/*.conf"
"/var/log/*.log"
"/var/log/**/*.log
"/path/to/logs/{app1,app2,app3}/data.log"
```
